# Date Formatter by Example


Examples:

| Date by Example         | => `strftime` Format |
|-------------------------|----------------|
| January 04, 2020        | => `%B %d, %Y` |
| Fri, Jan 04             | => `%a, %b %d` |
| 4 Jan 2020              | => `%-d %b %Y` |
| Friday, January 4, 2020 | => `%A, %B %-d, %Y` |
| Fri, Jan 04 8:00        | => `%a, %b %d %-H:%M` |
| 4 Jan 2020 08:00        | => `%-d %b %Y %H:%M` |
